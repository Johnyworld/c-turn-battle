# WW2 Hex Battle

2차 세계대전 배경의 헥스 격자 턴제 전략 게임입니다.
연합군 또는 추축군을 선택해 AI 상대와 1:1 전투를 벌입니다.

---

## 실행 방법

```bash
npm start
```

브라우저에서 `http://localhost:3000` 접속

> Node.js가 설치되어 있어야 합니다.

---

## 게임 플로우

```
[접속]
  │
  ▼
[진영 선택 화면]
  ├── 연합군 선택 → 플레이어: 연합군 / AI: 추축군
  └── 추축군 선택 → 플레이어: 추축군 / AI: 연합군
  │
  ▼
[전투 화면]
  │  플레이어 턴 → AI 턴 → 반복
  │
  ▼
[게임 종료]
  ├── 재도전 → 같은 진영으로 재시작
  └── 진영 재선택 → 선택 화면으로 복귀
```

---

## 조작 방법

| 조작 | 동작 |
|------|------|
| 아군 유닛 클릭 | 유닛 선택 (초록 = 이동 가능 범위 표시) |
| 이동 가능 헥스 클릭 | 선택 유닛 이동 |
| 적 유닛/기지 클릭 | 공격 (주황색 테두리 = 공격 가능) |
| 의무병 선택 후 아군 클릭 | 아군 회복 (초록 테두리 = 회복 가능) |
| **턴 종료** 버튼 | 현재 턴 종료 → AI 턴 자동 진행 |
| **FOG ON/OFF** 버튼 | 전장의 안개 켜기/끄기 |
| **새 게임** 버튼 | 같은 진영으로 재시작 |

## 유닛 정보

| 기호 | 유닛 | HP | 공격력 | 이동력 | 사거리 | 특성 |
|------|------|----|-------|-------|-------|------|
| **T** | 탱크 | 80 | 35 | 3 | 1 | — |
| **A** | 야포 | 40 | 50 | 1 | 3 | 이동 후 공격 불가 |
| **I** | 보병 | 50 | 20 | 2 | 1 | — |
| **AT** | 대전차보병 | 40 | 45 | 2 | 1 | 탱크 대미지 ×1.5 |
| **M** | 의무병 | 30 | 5 | 2 | 1 | 인접 아군 HP +15 회복 |

각 진영 초기 배치: 탱크×2, 야포×1, 보병×3, 대전차보병×1, 의무병×1 (총 8유닛)

---

## 맵 & 기지

- **맵 크기**: 17 × 11 헥스
- **연합군 기지 (★)**: 좌측 중앙
- **추축군 기지 (★)**: 우측 중앙
- **승리 조건**: 상대 기지 HP를 0으로 만들면 승리

---

## 전장의 안개 (Fog of War)

- 플레이어 유닛 시야 반경(3헥스) 합집합만 표시
- 시야 밖의 적 유닛·기지는 숨겨짐
- 플레이어 기지 주변은 항상 가시 영역
- 헤더의 **FOG ON/OFF** 버튼으로 토글 가능

---

## AI 행동 원칙

AI는 매 유닛 행동마다 300ms 딜레이를 두어 시각적으로 확인할 수 있습니다.

1. **의무병** — 인접 아군 HP가 50% 이하이면 우선 회복
2. **야포** — 사거리 내 적이 있으면 이동 없이 공격, 없으면 전진
3. **일반 유닛** — 공격 가능한 타깃 중 HP가 가장 낮은 대상 공격
4. **이동 후 재공격** — 공격 불가 시 적 기지 방향으로 이동 후 공격 재시도

## 프로젝트 구조

```

├── server.js              # 정적 파일 서버 (포트 3000)
└── public/
    ├── index.html         # 진영 선택 + 전투 화면

        ├── combat.js      # 전투 로직
        ├── render.js      # Canvas 렌더링
        ├── ui.js          # DOM 업데이트
        ├── ai.js          # AI 엔진
        └── fog.js         # Fog of War 시야 계산
```
